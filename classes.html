Classes#

This page documents all classes describing all types of elements available to use in a Geometrix Workfield instance.

Points#

class Geometrix.point.Point(x, y, id)#

Class that describes a point

Parameters:

  • x – x-coordinate of the point

  • y – y-coordinate of the point

  • connected_to[] – list of Point objects the point is connected to with a line

  • id – ID of the point

connect(point: Point)#

Function that adds another Point to this point’s connected_to[]. Not designed to connect Point objects, use Workfield.add_line() instead.

Parameters:

point – Point to connect to this point

Returns:

nothing

Lines#

class Geometrix.line.Line(point1, point2, id)#

Class that describes a line

Parameters:

  • point1 – first Point

  • point2 – point to connect the first Point to

  • id – ID of the line

Circles#

class Geometrix.circle.Circle(centre, radius, id)#

Class that describes a circle

Parameters:

  • centre – ID of the circle’s centre Point (point has to be created prior to creation of the circle)

  • radius – the circle’s radius

  • id – the ID of the circle

calc_area()#

Function to calculate a circle’s area

Returns:

area of the circle

calc_circumference()#

Function to calculate a circle’s circumference

Returns:

circumference of the circle
